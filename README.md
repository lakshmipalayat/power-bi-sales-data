# power-bi-sales-data

A Power BI sales data analytics project featuring interactive dashboards and insights for sales performance tracking.

## Overview

This repository contains a Power BI presentation file (`pbi_presentation.pbix`) with sales data visualizations and analytics dashboards designed to provide comprehensive insights into sales performance metrics.

## Contents

- **pbi_presentation.pbix** - Main Power BI file containing dashboards and data models for sales analysis

## Features

- Interactive sales dashboards
- Sales performance metrics and KPIs
- Data-driven insights and visualizations
- Comprehensive reporting capabilities

## Requirements

- **Power BI Desktop** - To open and edit the `.pbix` file
- **Power BI Service** (optional) - To share and collaborate on the reports online

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lakshmipalayat/power-bi-sales-data.git
   cd power-bi-sales-data
   ```

2. **Open the Power BI file**:
   - Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
   - Open `pbi_presentation.pbix` with Power BI Desktop

3. **Explore the dashboards**:
   - Navigate through the different report pages
   - Use interactive filters and visualizations
   - Drill down into specific metrics

## File Sizes

- `pbi_presentation.pbix` - ~3 MB

## Usage

Use this Power BI file to:
- Track and analyze sales trends
- Monitor key performance indicators
- Generate sales reports
- Share insights with stakeholders
