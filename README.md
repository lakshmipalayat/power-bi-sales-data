# power-bi-sales-data

A Power BI sales data analytics project featuring interactive dashboards and insights for sales performance tracking.

## Overview

This repository contains a Power BI presentation file (`pbi_presentation.pbix`) with sales data visualizations and analytics dashboards designed to provide comprehensive insights into sales performance. The **Retail Business Analytics Report** includes multiple interactive dashboards tracking sales, profit, customer behavior, and regional performance.

## Dashboard Features

### 1. Sales Analytics Dashboard
Key metrics and visualizations:
- **Total Sales**: $12.64M
- **Total Sales LY (Last Year)**: $8.34M
- **YoY Sales Growth**: 51.54%
- **Avg Sales Per Order**: $504.99
- **Sales by Category**: Technology ($4.7M), Furniture ($4.1M), Office Supplies ($3.8M)
- **Sales by Segment**: Home Office ($2.31M), Corporate ($3.82M), Consumer ($6.51M)

## Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lakshmipalayat/power-bi-sales-data.git
   cd power-bi-sales-data
   ```

2. **Open the Power BI file**:
   - Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/)
   - Open `pbi_presentation.pbix` with Power BI Desktop

3. **Explore the dashboards**:
   - Navigate through the different report pages (Sales, Customer & Regional, Profit)
   - Use interactive filters and visualizations
   - Drill down into specific metrics by category, segment, region, or time period
   - Filter by Year, Month, and Country for detailed analysis
